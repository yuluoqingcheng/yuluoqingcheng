# 🚘 OpenPilot Toolbox

<table>
<tr>
<td width="65%" valign="top">

**OpenPilot Toolbox** 是一款围绕 OpenPilot 设备开发的综合管理工具，主要面向 Comma C2、C3、OpenPilot、SunnyPilot 以及不同车机系统环境。

项目希望将复杂的 Linux 命令、SSH 操作、设备维护和参数配置流程，封装成更加直观、稳定、容易使用的图形化功能。

### ✨ 主要能力

* OpenPilot 设备发现、连接与管理
* 局域网 Wi-Fi 自动扫描和自动回连
* SSH 远程连接、命令执行与交互终端
* 文件、命令和脚本一键下发
* 一键下发与 SSH 终端联动执行
* 驾驶设置和高级调试参数管理
* 参数备份、恢复、分享与一键应用
* 参数市场、评分、评论、支持与反对
* 脚本扫描、编写、上传、审核和在线执行
* 脚本执行终端与历史输出查看
* 驾驶数据统计、安全评分与排行榜
* 驾驶数据云端备份与格式化后自动恢复
* 多进程扭矩参数和横向延时分析
* 自动生成扭矩散点图和延时分析图
* 自动计算并推荐强制扭矩参数
* 设备体检、深度内存优化和存储优化
* Comma C2 / C3 硬件与系统分支识别
* CP、DP、SP0.98、SP2025、SP2026 等分支适配
* 比亚迪车机实验模式与开机自启
* 外接设备与小蓝灯硬件联动管理
* 设备时间、时区和时间服务器自动同步

</td>
<td width="35%" valign="top">

### 🧩 工具定位

```text
设备管理
    +
参数调试
    +
系统维护
    +
数据分析
    +
硬件联动
    +
社区分享
```

### 🎯 项目目标

让不熟悉 Linux、SSH 和命令行的用户，也能够更加方便地维护和使用 OpenPilot 设备。

把复杂的技术隐藏在工具背后，把简单、清晰、稳定的体验留给用户。

</td>
</tr>
